About this IDS

This IDS has been developed as part of a project for the course "INSE6170 - Network Security Architecture and Management" of Concordia University

This project uses HTML, the standard markup language for creating web pages, along with the popular Bootstrap CSS framework to create a responsive and visually appealing user interface.

On the server-side, it uses Python, a versatile and powerful programming language, along with Flask, a lightweight web framework, to build the backend logic and handle dynamic content generation. For data storage, and employs PostgreSQL, to manage the relational database.
